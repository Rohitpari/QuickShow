# 🎬 QuickShow – Movie Ticket Booking Application

QuickShow is a full-stack movie ticket booking web application where users can browse movies, check show timings, select seats, and book tickets online.

---

## 🚀 Features

- 🎥 Browse latest movies
- 📅 View show timings
- 💺 Select seats
- 🛒 Book movie tickets
- 🔐 User authentication (Login / Signup)
- 📦 Backend API integration
- 💳 Booking management system

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
